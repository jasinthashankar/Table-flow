# TableFlow Design System

TableFlow uses a custom hospitality-operations visual language rather than a generic dashboard template.

## Direction

- **Editorial landing experience:** serif display typography and an original floor-operations console
- **Operational workspace:** dark role-aware sidebar, translucent top bar, structured data surfaces
- **Warm professional palette:** ink, parchment, cobalt, coral, sage, and muted gold
- **Information hierarchy:** compact labels, expressive metrics, restrained status colours
- **Motion:** subtle entrance, hover, and navigation feedback with reduced-motion support

## Core colours

- Ink: `#111827`
- Deep navy: `#172033`
- Cobalt: `#3657FF`
- Coral: `#FF6B5F`
- Sage: `#9BB9A2`
- Gold: `#E8B455`
- Paper: `#F4F1EB`
- Cloud: `#F8F9FB`

## Typography

- Interface: Manrope
- Display moments: DM Serif Display

## Product principles

1. Data remains readable before decoration.
2. Status is communicated with text and colour.
3. Guest and admin experiences share one system but use role-specific navigation.
4. The interface remains responsive and keyboard-friendly.
5. Decorative visuals are built with CSS and application data patterns, not stock dashboard illustrations.
